# Go-Universify — `.github` Repository

Welcome to the **Go-Universify** organization's `.github` repository!

## What is this repository?

This is a special repository that holds organization-wide [community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) and configuration. Files placed here serve as defaults for every repository in the **Go-Universify** organization that does not have its own version of those files.

## About Go-Universify

**Go-Universify** is a Go (Golang) based open-source initiative focused on building reusable, well-tested packages and tools for the Go ecosystem. Our goal is to make high-quality Go software universally accessible to developers of all experience levels.

## Community Health Files

This repository may contain any of the following default files that apply across all Go-Universify repositories:

| File | Purpose |
|------|---------|
| `CODE_OF_CONDUCT.md` | Standards for community behavior |
| `CONTRIBUTING.md` | Guidelines for contributing to our projects |
| `SECURITY.md` | How to responsibly report security vulnerabilities |
| `SUPPORT.md` | Where to get help |
| `FUNDING.yml` | Sponsorship and funding information |
| `profile/README.md` | Organization profile shown on the Go-Universify GitHub page |
